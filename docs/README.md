# Career Knowledge Base

Use this folder as the permanent source of truth for portfolio updates.

## Files

- resume-master.md: full resume structure and future master notes
- career-goals.md: current career direction and long-term goals
- learning-roadmap.md: learning priorities and roadmap
- achievements.md: achievements and professional highlights
- certifications.md: certifications and learning milestones

## How to update the portfolio

1. Update the relevant markdown file in docs/.
2. Keep the content files in content/ aligned with the same information.
3. Replace the resume PDF in public/resume/ when available.
4. Rebuild the site so the changes appear in the portfolio.
